1、能够说出常见集合的特点
​	Collection
​		List -- 有序，有索引，允许重复元素
​			ArrayList	-- 数组实现，查询快，增删慢
​			LinkedList	-- 链表实现，增删快，查询慢
​			Vector -- 数组实现，线程安全的，被ArrayList取代
                Stack -- 栈 LIFO
​		Set -- 无序，无索引，不允许重复元素
​			HashSet		-- 保证元素唯一，依赖哈希表（数组+链表+红黑树），需要重写hashCode和equals方法
​			TreeSet		-- 保证元素唯一，依赖排序（自然顺序、比较器顺序）
​			LinkedHashSet -- 由链表保证迭代顺序，由哈希保证元素不重复。
    Map
        HashMap
        TreeMap
        Hashtable -- 线程同步的Map集合 被HashMap 取代
